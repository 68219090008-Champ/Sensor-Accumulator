จ่ายไฟให้เซ็นเซอร์ : ต่อVCCขาหนึ่งเข้ากับแหล่งจ่ายไฟ 3.3V หรือ 5V และGNDต่อขาอีกข้างหนึ่งเข้ากับกราวด์
เชื่อมต่อเอาต์พุต : ต่อOUTพินเข้ากับพินอินพุตของไมโครคอนโทรลเลอร์ (เช่น Arduino) หรือต่อโดยตรงกับ LED/ออดสำหรับงานง่ายๆ
ปรับความไว (ถ้ามี) : เซ็นเซอร์บางตัวมีโพเทนชิโอมิเตอร์สำหรับปรับช่วงการตรวจจับ หมุนโพเทนชิโอมิเตอร์ตามเข็มนาฬิกาหรือทวนเข็มนาฬิกาเพื่อปรับความไวให้เหมาะสม
อ่านค่าเอาต์พุต : เซ็นเซอร์จะส่งสัญญาณ HIGH เมื่อตรวจพบวัตถุในระยะทำการ และส่งสัญญาณ LOW ในกรณีอื่น ๆ สำหรับเซ็นเซอร์แบบอนาล็อก แรงดันไฟฟ้าขาออกจะแปรผันตามระยะห่างของวัตถุ

ข้อควรพิจารณาที่สำคัญและแนวปฏิบัติที่ดีที่สุดข้อควรพิจารณาที่สำคัญและแนวปฏิบัติที่ดีที่สุด
หลีกเลี่ยงการรบกวนจากแสงโดยรอบ : แสงโดยรอบที่แรง เช่น แสงแดด อาจรบกวนการทำงานของเซ็นเซอร์ได้ ควรใช้เซ็นเซอร์ในสภาพแสงที่ควบคุมได้ หรือป้องกันไม่ให้แสงส่องโดยตรง
ตำแหน่งที่เหมาะสมที่สุด : ตรวจสอบให้แน่ใจว่าได้ติดตั้งเซ็นเซอร์ในมุมและระยะห่างที่เหมาะสมจากวัตถุเป้าหมายเพื่อการตรวจจับที่แม่นยำ
ความเสถียรของแหล่งจ่ายไฟ : ใช้แหล่งจ่ายไฟที่เสถียรเพื่อหลีกเลี่ยงการทำงานที่ผิดปกติ
